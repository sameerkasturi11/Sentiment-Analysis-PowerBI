# Sentiment Analysis Power BI Dashboard

This project analyzes customer reviews using Python and visualizes the results using Power BI.

## Tools Used
- Python
- Pandas
- TextBlob
- Power BI
- Excel

## Project Structure

dataset → contains customer reviews dataset  
python_script → sentiment analysis code  
powerbi_dashboard → Power BI dashboard  
screenshots → dashboard image

## Insights
The dashboard shows:
- Sentiment distribution
- Product level sentiment
- Review trends over time
